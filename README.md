# ReservaPro Studio

Aplicacion web para administrar reservas, agenda, servicios, ingresos diarios y clientes prioritarios de un estudio profesional.

## Descripcion

ReservaPro Studio esta disenado como una pieza de portafolio para demostrar producto, UI, gestion de estado en frontend y persistencia local. La app permite crear, filtrar y cancelar reservas, calcular ingresos del dia y mostrar un catalogo de servicios.

## Funcionalidades

- Formulario completo para crear citas.
- Agenda con filtros por todas, hoy y VIP.
- Cancelacion de reservas.
- Persistencia con LocalStorage.
- Resumen automatico de citas, ingresos y ocupacion.
- Catalogo de servicios con duracion, precio y demanda.
- Modulo CRM con clientes prioritarios.
- Interfaz responsive con enfoque comercial.

## Tecnologias

- HTML5
- CSS3
- JavaScript vanilla
- LocalStorage API

## Como ejecutar

Abre `index.html` directamente en el navegador. No necesita dependencias ni servidor.

## Valor para portafolio

Este proyecto muestra capacidad para construir una herramienta de negocio usable, con flujos reales, datos persistentes y una presentacion visual profesional.
